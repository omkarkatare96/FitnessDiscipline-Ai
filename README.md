🧠 FitDiscipline AI — Smart Fitness Web App

**FitDiscipline AI** is an intelligent fitness platform designed to help users **train smarter, recover better, and stay consistent**.
It combines **AI coaching, recovery analytics, workout planning, and nutrition guidance** to act like a personal digital fitness assistant.

---

🚀 Features

* 🏋️ **Muscle Trainer** with exercise images, GIFs, and video guidance
* 📊 **Readiness & Recovery System** with 7 health metrics and BMR calculation
* 🔀 **Smart Reschedule Engine** that merges missed workouts intelligently
* 🧠 **AI Fitness Coach** for workout advice and injury recovery guidance
* 🥗 **Nutrition Hub** with diet tips and nutrition videos
* ▶ **Workout Library** with curated YouTube training videos
* 🔔 **Smart Notifications** for workouts, AI responses, and readiness insights
* ☁️ **Firebase Backend** with secure user authentication and database

---

🧩 Tech Stack

**Frontend:** React, TypeScript, Vite, Tailwind CSS
**Backend:** Firebase Authentication, Firestore
**AI Integration:** OpenRouter API (Mistral AI)
**Charts & Analytics:** Recharts

---

⚙️ Running the Project

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open in browser:

```
http://localhost:5173
```
